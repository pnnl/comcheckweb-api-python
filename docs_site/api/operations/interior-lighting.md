# Interior Lighting

!!! warning "Under Development"
    Interior lighting operations are not yet implemented. This section is planned for a future release.

This page will cover operations for managing interior lighting in COMcheck projects, including:

- Adding and updating whole building use areas with lighting data
- Managing activity use areas and fixture schedules
- Lighting power allowance calculations
- Lighting controls

The underlying types for interior lighting (`WholeBldgUse`, `ActivityUse`, `Fixture`, `FixtureSchedule`) are already available. See the [Types Guide](types-guide.md) for details on importing and using them.
